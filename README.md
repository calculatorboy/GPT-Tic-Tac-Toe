# GPT-Tic-Tac-Toe
This is a repository to archive the results of my chat GPT conversations. I was trying to get it to make a tic tac toe game, but it kept throwing network errors before it could even get close to done. I tried a few methods that didn't work until I stumbled upon the method of separating the game into multiple messages by line length. Though it doesn't solve the core problem, it works okay and with a little coaxing when it goes in the wrong direction it is usually fine.

I made two versions. The raw code that came out of the chatbot and a polished version. The raw one works it just is a little clunky. The polished version adds the bots how to, changes some prompts so they are in the correct orders, and moves the board down a line so it doesn't become offset by the prompt.
